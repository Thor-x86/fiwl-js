## What should I know before I get started?

### FIWL Architecture

A typical FIWL.js project has 3 main files:

- `main.fiwl`
- `style.varl`
- `manifest.json`

To know more about the above files and how FIWL works check out the [docs](https://fiwl-js.github.io)
